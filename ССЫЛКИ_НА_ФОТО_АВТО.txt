================================================================
   КАК ЗАГРУЗИТЬ ФОТО В АДМИНКУ
================================================================

ШАГ 1. Скачай нужную картинку:
  - Открой URL в браузере
  - Правый клик на картинке → "Сохранить как..."
  - Сохрани на рабочий стол, например "kia_rio.jpg"

ШАГ 2. Загрузи в админку:
  - http://127.0.0.1:8000/admin/
  - Войди как admin / admin12345
  - "Автомобили" → выбери нужное авто
  - В секции "📸 Фотография" → поле "Main image" → Choose File
  - Выбери скачанную картинку
  - "Сохранить"

ШАГ 3. Добавь доп. фото для галереи (опционально):
  - Внизу страницы редактирования авто
  - Секция "Дополнительные фото"
  - Загрузи 2-4 картинки с разных ракурсов


================================================================
   ИЗМЕНИТЬ РАЗМЕР ФОТО ИЗ URL
================================================================

Можно менять параметры прямо в URL Unsplash:

  - Ширина 1200px: ?w=1200&q=80
  - Ширина 600px:  ?w=600&q=80
  - HD качество:   ?w=1200&q=90
  - Малое превью:  ?w=300&q=70

Пример:
  https://images.unsplash.com/photo-1549399542-7e3f8b79c341?w=1200&q=90
