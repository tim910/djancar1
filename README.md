# 🚗 DjanCar — каршеринг в Ульяновске

Современный веб-сайт каршеринговой компании "DjanCar" для города Ульяновск.
Дипломный проект. Полнофункциональная платформа аренды авто с интерактивной картой,
профилями пользователей, тарифами, отзывами и админкой.

## 🛠 Стек технологий

| Слой        | Технологии                                |
|-------------|-------------------------------------------|
| **Backend** | Python 3.10+, Django 5.0, Django REST Framework |
| **БД**      | PostgreSQL 14+                            |
| **Frontend**| HTML5, CSS3 (custom, без фреймворков), Vanilla JS |
| **Карты**   | Яндекс.Карты API                          |
| **Прочее**  | Pillow, python-decouple, django-widget-tweaks |

## 🚀 Запуск проекта

### 1. Установка зависимостей

```bash
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS/Linux
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Скопируйте `.env.example` в `.env` и заполните:

```env
SECRET_KEY=ваш-длинный-секретный-ключ
DEBUG=True
DB_NAME=djancar
DB_USER=postgres
DB_PASSWORD=ваш_пароль
DB_HOST=127.0.0.1
DB_PORT=5432
YANDEX_MAPS_API_KEY=ваш_ключ_с_developer.tech.yandex.ru
```

### 3. Создание базы данных в PostgreSQL

```sql
-- Подключитесь к psql под пользователем postgres
CREATE DATABASE djancar;
CREATE USER djancar_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE djancar TO djancar_user;
```

### 4. Миграции и фикстуры

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py seed_demo              # загрузит 24 авто, 12 зон, тарифы, юзеров
python manage.py collectstatic --noinput
python manage.py runserver
```

Откройте http://127.0.0.1:8000

### 5. Тестовые учётные записи

| Логин   | Пароль       | Роль          |
|---------|--------------|---------------|
| admin   | admin12345   | Суперюзер     |
| demo    | demo12345    | Обычный юзер  |

### 🛟 Быстрый запуск без PostgreSQL

В `djancar/settings.py` закомментируйте блок `postgresql` и раскомментируйте `sqlite3`.

## 🗺 Получение ключа Яндекс.Карт

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите через аккаунт Яндекс.
3. Подключите тариф "JavaScript API и HTTP Геокодер" (бесплатно до 25 000 запросов в день).
4. Скопируйте API-ключ в `.env` → `YANDEX_MAPS_API_KEY`.

Без ключа карта работать не будет — будет белый прямоугольник.
